API 文档

这是一个基于Vercel Serverless Functions的邮件发送API服务。

API端点

POST /api

请求参数


        {
        "to": "收件人邮箱地址",
        "subject": "邮件主题",
        "text": "纯文本内容（可选，html 和 text 必须提供一个）",
        "html": "HTML格式内容（可选，html 和 text 必须提供一个）",
        "from_name": "发送者名称（可选）"
        }

响应格式

成功：

{
  "success": true,
  "messageId": "邮件ID"
}

失败：

{
  "error": "错误信息",
  "details": "详细错误信息"
}
